export_dd
===============

export_data_dictionary

> export_dd的运行环境要求PHP5.4以上。

## 目录结构

初始的目录结构如下：

~~~
www  WEB部署目录
├─config                配置目录
│  ├─config.php         公共配置文件
│  └─database.php       数据库配置文件
│
├─template              模板目录     用于apache的重写
├─extend                扩展类库目录         授权说明文件
├─README.md             README 文件
├─index.php             执行文件
~~~

## 使用
在index.php目录执行 php index.php
